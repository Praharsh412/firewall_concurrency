Clientside code:
Description of the tasks done by the client program:
  • Creation of two threads, one for receiving and other for sending the information which runs parallelly.
  • The client would be receiving the messages sent to the server from various other clients.
Working of the code:
  • At each client machine this code has to be compiled and executed to connect to the server.
  • The compilation of the code is done by the command:
    g++ -o client client.cpp -lpthread.
  • The execution is done by specifying the ip address and the port number of the server to which the client is trying to connect.
    ./client 127.0.0.1 30666

It has two functions one for receiving and one for sending the information.

Now the threads are created for send and receive functions respectively and later joinded to the main process.

while (thread_num <3 )
{

for(int i = 0; i < 3; i++)
